Project Overview

The Real-Time Weather Forecast Dashboard is a fully interactive Power BI report powered by a live Weather API, offering a deep dive into current weather conditions, 14-day forecasts, and air quality indexes for selected cities across India.

Built for clarity and engagement, this dashboard allows users to monitor real-time environmental metrics, assess air pollution levels, and plan activities based on weather and precipitation insights — all dynamically controlled through slicers, dropdowns, and bookmarks.

Business Impact

  • 80% faster weather data analysis compared to manual methods
  • 30% improvement in decision-making for outdoor event planning
  • Reduced risk by monitoring air quality for health-sensitive activities
  • Increased engagement through intuitive visualizations and interactivity

Key Features

Dynamic City Selection

Button slicers for 5+ Indian cities and 1 canadian city with real-time temperature and weather status updates. Includes favorites system for quick access.

14-Day Forecast

Interactive line chart with temperature trends, anomaly detection, and comparison tools. Custom date range selection available.

Rain Probability

Probability analysis with 3-hour granularity. Includes precipitation amount forecasts and storm warnings.

Air Quality Index

Comprehensive pollution monitoring with health recommendations based on current AQI levels and pollutant concentrations.

Advanced Metrics

UV index with sun protection advice, humidity comfort levels, wind chill calculations, and barometric pressure trends.

Cross-Platform

Responsive design works on all devices. Mobile-optimized layout available through Power BI service app.

Data Architecture

  1. API Integration: Live connection to WeatherAPI.com with automatic refresh every 15 minutes
  2. Data Transformation: Power Query cleanses and structures nested JSON into relational tables
  3. Data Model: Star schema with fact tables for measurements and dimension tables for locations/time
  4. Calculations: 50+ DAX measures for derived metrics and KPIs
  5. Visual Layer: Interactive reports with drill-through capabilities
  6. Deployment: Published to Power BI Service with scheduled refreshes
